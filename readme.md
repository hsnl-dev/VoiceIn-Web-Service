VoiceIn API Service
==
API Web Service using jersey on GAE.

Prerequisite
==
- Google App Engine SDK
- JRE 1.8
- JAVA 1.8

Build
==
```sh
mvn clean install
```
Run
==
Use google dev server
```sh
mvn appengine:devserver
```

Deploy
==
```sh
mvn appengine:update
```
